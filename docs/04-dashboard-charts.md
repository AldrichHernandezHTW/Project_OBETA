# 04. Dashboard Charts

This document explains each chart used in the OBETA Warehouse Picking Performance Dashboard.

---

## 1. Picking Activity Over Time

### Chart Type

Line chart

### Fields Used

| Field | Purpose |
|---|---|
| `year` | Time grouping |
| `month` | Time grouping |
| `pick_id` | Count of picking operations |

### Calculation

```sql
SELECT
    year,
    month,
    COUNT(pick_id) AS number_of_picks
FROM obeta_production.pick
GROUP BY year, month
ORDER BY year, month;
```

### Meaning

This chart shows how picking activity changed month by month from 2011 to 2020.

### Why We Chose It

It helps identify trends, seasonality, growth, drops, or unusual periods in warehouse activity.

### Business Question Answered

```text
Did warehouse picking activity increase, decrease, or remain stable over time?
```

## 2. Top Product Groups by Picks

### Chart Type

Stacked bar chart

### Fields Used

| Field | Purpose |
|---|---|
| `product_group` | Product category |
| `year` | Time grouping |
| `pick_id` | Count of picks |

### Calculation

```sql
SELECT
    p.year,
    pr.product_group,
    COUNT(p.pick_id) AS number_of_picks
FROM obeta_production.pick p
JOIN obeta_production.product pr
    ON p.product_key = pr.product_key
GROUP BY p.year, pr.product_group
ORDER BY p.year, number_of_picks DESC;
```

### Meaning

This chart shows which product groups generated the most picking operations.

### Why We Chose It

It helps identify the product categories that create the highest warehouse workload.

### Business Question Answered

```text
Which product groups are picked most often?
```

## 3. Average Duration by Origin

### Chart Type

Horizontal bar chart

### Fields Used

| Field | Purpose |
|---|---|
| `origin` | Order origin, such as `CUSTOMER` or `STORE` |
| `order_duration` | Time needed to complete an order |

### Calculation

```sql
SELECT
    o.origin,
    AVG(p.order_duration) AS average_order_duration
FROM obeta_production.pick p
JOIN obeta_production.`order` o
    ON p.order_key = o.order_key
GROUP BY o.origin
ORDER BY average_order_duration DESC;
```

### Meaning

This chart compares average order processing time by origin.

### Why We Chose It

It helps detect if customer orders or store orders are slower to process.

### Dashboard Interpretation

In the dashboard, `CUSTOMER` orders take longer on average than `STORE` orders. This may indicate that customer orders are more complex, smaller, more fragmented, or require more picking effort.

### Business Question Answered

```text
Are customer orders or store orders slower to process?
```

## 4. Pick Volume by Warehouse Section

### Chart Type

Horizontal bar chart

### Fields Used

| Field | Purpose |
|---|---|
| `warehouse_section` | Warehouse section |
| `pick_volume` | Total volume handled |

### Calculation

```sql
SELECT
    pr.warehouse_section,
    SUM(p.pick_volume) AS total_pick_volume
FROM obeta_production.pick p
JOIN obeta_production.product pr
    ON p.product_key = pr.product_key
GROUP BY pr.warehouse_section
ORDER BY total_pick_volume DESC;
```

### Meaning

This chart shows which warehouse sections handled the highest picking volume.

### Why We Chose It

It helps identify the busiest warehouse sections and possible operational bottlenecks.

### Business Question Answered

```text
Which warehouse sections carry the largest operational load?
```

## 5. Picks by Origin

### Chart Type

Horizontal bar chart

### Fields Used

| Field | Purpose |
|---|---|
| `origin` | Order origin |
| `pick_id` | Count of picks |

### Calculation

```sql
SELECT
    o.origin,
    COUNT(p.pick_id) AS number_of_picks
FROM obeta_production.pick p
JOIN obeta_production.`order` o
    ON p.order_key = o.order_key
GROUP BY o.origin
ORDER BY number_of_picks DESC;
```

### Meaning

This chart compares how many picking operations were generated by each origin.

### Why We Chose It

It helps understand where most warehouse workload comes from.

### Dashboard Interpretation

Most picks come from `CUSTOMER` orders, while `STORE` orders represent a smaller part of the total picking activity.

### Business Question Answered

```text
Are most picks generated by customer orders or store orders?
```
